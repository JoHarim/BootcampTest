# S1. 생년월일 입력 — 내 사주 정보를 받아 저장

## 들어오는 길
- 앱 첫 진입 시 저장된 생일이 없으면 이 화면으로 온다.
- S2 오늘의 운세 화면에서 "정보 수정" 버튼을 누르면 이 화면으로 온다.

## 화면 구성 (위에서 아래로)
1. 제목 "내 사주 정보" → 2. 생년월일 입력칸(필수) → 3. 태어난 시각 입력칸(선택) → 4. 저장 버튼
(02-spec.md의 S1 표와 일치)

## 행동과 흐름 (버튼·입력마다 한 줄)
| 행동 | 성공하면 | 실패하면 |
|---|---|---|
| 저장 버튼 | 정보를 브라우저에 저장 → S2 오늘의 운세로 이동 | 입력 오류 안내 표시, 화면 유지 |

## 예외 5종 (전부 답해야 이 화면이 완성)
| 상황 | 이 화면의 처리 (문구까지) |
|---|---|
| 비어 있을 때 (데이터 0건) | 해당 없음 (이 화면은 새로 입력하는 곳) |
| 로딩 중 | 해당 없음 (즉시 렌더, 통신 없음) |
| 실패 (저장 에러) | "저장에 실패했어요. 다시 눌러주세요" 안내 |
| 권한 없음 · 비로그인 | 해당 없음 (로그인 없음) |
| 잘못된 입력 | 생년월일 비었으면 "생년월일을 입력해주세요" / 미래 날짜·없는 날이면 "올바른 생년월일을 입력해주세요" · 시각은 비워도 됨 |
