# S1 타이틀

## 진입 경로
- 사이트 루트(/) 첫 진입
- S3 결과 화면의 "타이틀로" 버튼

## 보이는 것
- 게임 로고 (텍스트 아트: LUCKY RUN + 코인·슬롯 장식)
- 새 런 시작 버튼 (메인 CTA)
- 이어하기 버튼 — 저장된 런이 있을 때만, 진행 상황 요약과 함께 (예: "라운드 3 · 1,240코인")
- 최고 기록 배지: 최다 라운드 · 최고 코인 · 누적 런 수 (기록이 있을 때만)
- 기록 초기화 (작은 텍스트 버튼, 기록 있을 때만)
- 하단 고지: "본 게임의 코인은 가상 재화로 현금 교환·환전이 불가능합니다"
- 우상단 음소거 토글

## 행동 / 흐름
- 새 런: 진행 중 런이 없으면 즉시 런 시작 → S2. 있으면 "지금 진행 중인 런을 버립니까?" 확인 후 시작.
- 이어하기: 저장 상태 그대로 S2.
- 기록 초기화: "정말 지울까요? 되돌릴 수 없습니다" 확인 → BEST_RECORD 삭제.

## 예외 5종
- 비었을 때: 기록 없음 → 기록 배지·초기화 버튼 숨김. 저장 런 없음 → 이어하기 숨김.
- 로딩: 마운트 전 저장소 읽는 동안 버튼 대신 중립 화면(하이드레이션 불일치 방지) — 사주 플래너와 같은 패턴.
- 실패: 저장 데이터 파싱 실패 → 해당 키 없음 취급(조용히), 에러 표시 없음.
- 권한 없음: 해당 없음(로그인 없음).
- 잘못 입력: 입력 필드 없음 — 해당 없음.

## 데이터 연결
- 읽기: luckyrun:best, luckyrun:run, luckyrun:settings (저장 어댑터 경유)
- 쓰기: 기록 초기화 시 best 삭제, 새 런 시작 시 run 초기화
