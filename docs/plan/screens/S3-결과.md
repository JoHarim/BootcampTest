# S3 결과 (게임오버)

## 진입 경로
- S2 에서 기회 소진+목표 미달, 또는 파산

## 보이는 것
- "게임 오버" 헤드라인 + 게임오버 사유 한 줄 (목표 미달 / 파산)
- 도달 라운드 (크게), 최종 코인, 이번 런 통계(플레이 횟수·최대 당첨)
- 신기록이면 "신기록!" 배지 + 컨페티
- 최고 기록 비교 (최다 라운드 · 최고 코인)
- 새 런 버튼 (메인 CTA), 타이틀로 버튼

## 행동 / 흐름
- 진입 순간: run 삭제, best 갱신(더 좋을 때만), totalRuns +1.
- 새 런 → 즉시 런 시작 → S2. 타이틀로 → S1.

## 예외 5종
- 비었을 때: 통계가 0이어도 그대로 표시 (숨기지 않음).
- 로딩: 없음 (S2 에서 상태를 들고 넘어옴).
- 실패: best 저장 실패 시 화면 표시는 정상, 콘솔 경고만.
- 권한 없음: 해당 없음.
- 잘못 입력: 해당 없음. 단 S3 URL 직접 진입(새로고침)이면 런 데이터가 없으므로 S1 로 보낸다.

## 데이터 연결
- 쓰기: best 갱신, run 삭제
- 읽기: best (비교 표시용)
