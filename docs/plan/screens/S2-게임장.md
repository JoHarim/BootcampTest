# S2 게임장

## 진입 경로
- S1 새 런 / 이어하기

## 보이는 것
- 상단 HUD (상시 고정): 코인(카운트업 연출) · 라운드 번호 · 목표 코인(달성 시 체크 표시) · 남은 기회(10칸 게이지) · 음소거
- 부스 탭 2개: 스크래치 부스 / 슬롯머신 부스 (화면 이동 없이 전환)
- 스크래치 부스: 티켓 등급 3개 카드(동 50 / 은 150 / 금 400, 상금표 요약), 구매하면 티켓 등장 → 긁기 캔버스
- 슬롯 부스: 슬롯머신 캐비닛(마퀴 조명, 릴 창 3개 — 위아래 심볼이 살짝 보임, 페이라인, 우측 레버), 베팅 선택(50/150/400), 레버(또는 스핀 버튼)
- 라운드 클리어 팝업: "ROUND N CLEAR" + 남은 코인 이월·다음 목표 → 다음 라운드 버튼

## 행동 / 흐름
- 플레이 1회 = 티켓 1장 구매 또는 스핀 1회. 코인 차감 → 기회 1 소모 → 결과 → 코인 지급 → 저장.
- 긁기: 포인터로 45% 이상 긁으면 자동 공개. 당첨이면 효과음+컨페티, 금액 카운트업.
- 슬롯: 릴 3개 순차 정지(1.3/2.1/3.1초). 니어미스(7-7-X) 연출. 7 트리플 = 잭팟(팡파레+화면 흔들림+컨페티 3연발).
- 목표 조기 달성: HUD 옆 "라운드 마감" 버튼 활성 — 누르면 남은 기회당 보너스 코인(25 x 라운드) 지급 후 클리어 팝업. 계속 플레이하다 목표 밑으로 떨어지면 버튼은 다시 꺼진다.
- 기회 소진 판정: 목표 달성 → 클리어 팝업(목표 x1.6 반올림, 기회 10 리셋, 코인 이월) / 미달 → S3.
- 파산 판정: 코인이 50 미만이고 목표 미달이며 진행 중 연출이 없으면 즉시 S3.

## 미스터 핀 대출 (2026-07-08 변경 제안으로 추가)
- HUD 우측에 "한탕 자금?" 버튼(대출 가능 시) / "전액 상환 -N" 버튼(빚 활성 시). 빚 활성 중 코인 아래 빚 배지(+N/기회)와 순자산 병기, 목표 게이지는 순자산 기준.
- 대출 모달: 대사로 금액·이자·조건 고지 후 [도장 꽝] / [사양]. 구제 모달: 파산 직전 1회, 총 상환·필요 금액 표기, 거절 = 게임오버.
- 라운드 클리어 팝업에 "미스터 핀 몫 회수 -N" 줄 추가(빚 있었을 때).

## 예외 5종
- 비었을 때: 코인이 티켓/베팅 가격보다 적으면 해당 버튼 비활성 + "코인 부족" 표시.
- 로딩: 릴 회전·긁기 중에는 다른 플레이 버튼 잠금 (이중 차감 방지).
- 실패: 저장 실패(용량 등)해도 게임은 계속 — 이어하기만 안 될 수 있음, 콘솔 경고만.
- 권한 없음: 해당 없음.
- 잘못 입력: 버튼 선택형만 존재. 연출 중 연타는 잠금으로 무시.
