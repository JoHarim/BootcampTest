# 시스템 지도 — 럭키런

```
┌─ 벽 안 (우리가 만드는 것) ─────────────────────────────┐
│                                                      │
│  홀 (화면, apps/web/app/)                             │
│    S1 타이틀  ·  S2 게임장(스크래치 부스/슬롯 부스)  ·  S3 결과 │
│    사주 플래너(완료작)는 /saju 로 이사                     │
│                                                      │
│  주방 (서버): 없음 — 전부 브라우저에서 계산               │
│    게임 규칙·확률표: apps/web/lib/game/rules.ts          │
│    런 상태 머신:      apps/web/lib/game/run.ts           │
│                                                      │
│  창고 (저장): localStorage 3키                          │
│    접근은 저장 어댑터(lib/game/save.ts)로만 — 스팀판 대비    │
│                                                      │
└──────────────────────────────────────────────────┘
     여는 문 (외부 서비스)
       1. Netlify — 정적 배포 (새 사이트, 게임 전용 URL)
     안 여는 문
       Supabase(로그인·DB) · 결제 · 광고 · AI · 구글폰트 — 이번 MVP 없음
```

| 블럭 | 내용 |
|------|------|
| 공간 | 게임 웹 1개 (화면 3장) + 보존용 /saju |
| 역할 | 익명 방문자 1종 (관리자 없음) |
| 창고 | localStorage 3키 (best / run / settings), 저장 어댑터 경유 |
| 문 | Netlify 배포 1개. 그 외 0 |

## 스택
- Next.js (App Router) 정적 export — 기존 밀키트 그대로
- 그래픽: CSS + Canvas(긁기) + 이모지 심볼. 외부 에셋 0
- 사운드: Web Audio 합성 (파일 없음)
- 스팀판(로드맵): 같은 코드를 Electron 또는 Tauri 로 감싸고 저장 어댑터만 교체
