# 유저 플로우 — 사주 플래너 (실습)

## 메인 플로우 (첫 방문 → 목적 달성)
```mermaid
flowchart TD
  A[앱 접속] --> B{저장된 생일 있나?}
  B -->|없음| C[S1 생년월일 입력 화면]
  B -->|있음| F[S2 오늘의 운세 화면]
  C --> D[생년월일 입력, 시각은 선택]
  D --> E{저장 버튼}
  E -->|입력 OK| G[브라우저에 저장]
  E -->|빈 값·이상한 날짜| C2[오류 안내 + 다시 입력]
  C2 --> D
  G --> F
  F --> H[오늘 일진 + 등급 + 한 줄 표시]
  H --> I[정보 수정 버튼 → S1]
  I --> C
```

## 재방문 플로우
- 다음에 다시 들어오면 저장된 생일을 읽어 곧바로 S2 오늘의 운세를 보여준다 (생년월일 재입력 없음).

## 운영자 플로우
- 없음 (단일 사용자, 관리자 페이지 없음)

## 화면 색인 (상세는 화면별 파일로)
| 화면 | 파일 | 목적 한 가지 |
|---|---|---|
| S1 생년월일 입력 | screens/S1-생년월일입력.md | 내 사주 정보를 받아 저장 |
| S2 오늘의 운세 | screens/S2-오늘의운세.md | 저장된 사주로 오늘 운세 표시 |
