# 기능명세서 — 사주 플래너 (실습)

## 화면: S1 생년월일 입력 (내 사주 정보를 받아 저장한다)
| No | 구분 | 기능 | 설명 | 우선순위 |
|---|---|---|---|---|
| 1 | 표시 | 제목 | "내 사주 정보" | MVP |
| 2 | 입력 | 생년월일 | 필수 · 날짜 선택 · 비면 "생년월일을 입력해주세요" | MVP |
| 3 | 입력 | 태어난 시각 | 선택 · 모르면 비워둠(비우면 시주 제외) | MVP |
| 4 | 동작 | 저장 버튼 | 누르면 정보를 브라우저에 저장 → S2 오늘의 운세로 이동 | MVP |

## 화면: S2 오늘의 운세 (저장된 사주로 오늘 운세를 보여준다)
| No | 구분 | 기능 | 설명 | 우선순위 |
|---|---|---|---|---|
| 1 | 표시 | 오늘 날짜 | 상단에 오늘 날짜 | MVP |
| 2 | 표시 | 오늘의 일진 | 예: "오늘은 갑자(甲子)일" (lunar-javascript로 계산) | MVP |
| 3 | 표시 | 운세 등급 | 좋음 / 보통 / 주의 (내 일간 vs 오늘 일진의 오행 상생·상극 + 십신 보정) | MVP |
| 4 | 표시 | 한 줄 코멘트 | 오늘 일진의 십신(예: 정재·편관 등)에 맞춘 짧은 문구 | MVP |
| 5 | 표시 | 면책 | "운세는 재미로 참고해주세요" 작게 | MVP |
| 6 | 동작 | 정보 수정 | 누르면 → S1 입력 화면 | MVP |
| 7 | 표시 | 정보 없을 때 | 저장된 생일이 없으면 자동으로 S1 입력 화면으로 보냄 | MVP |
