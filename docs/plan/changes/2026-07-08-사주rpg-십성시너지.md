# 변경 제안서 — 사주 RPG 십성 시너지

> 승인: 2026-07-08 회장 지시 "십성 ㄱㄱ" (세션 내 직접 요청)

## ① 뭘 바꾸나 (한 줄)
사주 7글자(일간 제외, 지지는 본기)의 십성을 5그룹(비겁·식상·재성·관성·인성)으로 집계해, 개수만큼 전투 보너스를 주는 시너지 층을 얹는다 — PRD 5-4 "동일 십성이 많이 모일수록 강화".

## ② 왜
모드 A에는 "내 사주의 성격이 전투 스타일이 된다"는 정체성을, 모드 B에는 "어떤 글자를 장착하느냐"의 빌드 전략(식상 몰빵 스킬 빌드, 관성+인성 탱크 빌드 등)을 준다. 수집의 후반 콘텐츠(글자 교체 최적화)에 목적이 생긴다.

## ③ 영향
- 계약: Character.tenGodCounts, BattleState.synergy, SynergyDef (types.ts)
- 데이터: content.ts SYNERGIES (수치·이름 단일 소스 — battle·UI 공용)
- 전투: 비겁=기본공격 +6%/개, 식상=스킬 +8%/개, 재성=치명 +3%p/개(상한 50), 관성=피격 −4%/개(하한 ×0.6), 인성=매턴 maxHp 1%/개 회복
- 화면: reveal·home에 시너지 카드, battle에 컴팩트 표기
- 저장: 변화 없음 (counts는 파생값 — 캐릭터 생성 시 재계산)
