# S2. 오늘의 운세 — 저장된 사주로 오늘 운세를 보여준다

## 들어오는 길
- 앱 진입 시 저장된 생일이 있으면 이 화면으로 온다.
- S1 입력 화면에서 저장 버튼을 누르면 이 화면으로 온다.

## 화면 구성 (위에서 아래로)
1. 오늘 날짜 → 2. 오늘의 일진(간지) → 3. 운세 등급(좋음/보통/주의) → 4. 한 줄 코멘트 → 5. 면책 한 줄 → 6. 정보 수정 버튼
(02-spec.md의 S2 표와 일치)

## 행동과 흐름 (버튼·입력마다 한 줄)
| 행동 | 성공하면 | 실패하면 |
|---|---|---|
| 화면 진입 | 저장된 생일로 오늘 일진 계산 → 등급·코멘트 표시 | 계산 실패 시 "운세를 불러오지 못했어요. 다시 시도" |
| 정보 수정 버튼 | S1 입력 화면으로 이동 | — |

## 예외 5종 (전부 답해야 이 화면이 완성)
| 상황 | 이 화면의 처리 (문구까지) |
|---|---|
| 비어 있을 때 (저장된 생일 없음) | 자동으로 S1 입력 화면으로 보냄 |
| 로딩 중 | 계산은 즉시라 로딩 거의 없음. 필요 시 "운세 계산 중…" 잠깐 표시 |
| 실패 (계산 에러) | 앱이 죽지 않고 "운세를 불러오지 못했어요. 다시 시도" + 정보 수정 버튼 |
| 권한 없음 · 비로그인 | 해당 없음 (로그인 없음) |
| 잘못된 입력 | 해당 없음 (입력 없는 화면). 저장된 생일이 이상하면 S1로 보내 다시 입력 |

## 읽고 쓰는 데이터 (창고 연결)
- 읽기: profile 표(localStorage `sajuweb:profile`)에서 `birthDate`·`birthTime`
- 쓰기: 없음
- 계산: lunar-javascript로 (1) 오늘 날짜의 일진 간지 (2) 내 생일의 일간(日干) → (a) 두 오행의 상생·상극으로 등급 판정 (b) 오늘 일진 천간의 십신(일간 기준, 음양 포함)을 뽑아 코멘트·등급 보정
