# 결정 기록 (ADR) — 럭키런

> 결정마다 4줄: 결정 / 왜 / 버린 대안 / 영향. 사주 플래너의 결정 기록은 docs/plan/archive-saju/06-decisions.md.

## 2026-07-07 게임을 메인(/)으로, 사주 플래너는 /saju 보존
- 결정: 게임이 루트 화면이 되고 사주 플래너는 /saju 로 이동한다.
- 왜: 이 레포의 주인공을 게임으로 교체 (회장 지시). 완료작은 지우지 않고 보존.
- 버린 대안: 게임을 /game 에 두기(주인공이 아님), 사주 삭제(작업물 유실).
- 영향: app/page.tsx 가 게임이 되고 app/saju/ 가 생긴다. 기존 Netlify 사주 배포는 재배포 전까지 영향 없음.

## 2026-07-07 BM: 리필·광고 없이 스팀 유료 게임 목표
- 결정: 코인 리필·광고 보상 없음. 처음부터 스팀 발매를 목표로 설계한다.
- 왜: 회장이 스팀 방향 선택. 무료 리필은 긴장감과 유료 판매 가치를 죽인다.
- 버린 대안: 광고 보상형(웹 킬링타임 BM), 시간 충전, 코인 IAP(사행성 규제 리스크).
- 영향: 파산이 의미를 가져야 하므로 런 구조가 필수가 됐다. 로드맵에 스팀 층(v2 시너지, v3 패키징) 추가.

## 2026-07-07 런 구조: 스테이지 목표형 (발라트로식)
- 결정: 라운드마다 "기회 10회 안에 목표 코인 달성", 실패·파산 = 런 종료.
- 왜: 순수 슬롯 시뮬은 스팀에서 안 팔린다 — 목표와 제한이 긴장·전략(베팅 크기 선택)을 만든다.
- 버린 대안: 자유 생존형(긴장 곡선이 평평함).
- 영향: HUD 에 라운드·목표·남은 기회 표시, 라운드 클리어 팝업, 목표 상승 곡선 밸런스 필요.

## 2026-07-07 저장: localStorage + 저장 어댑터 한 겹
- 결정: 저장은 localStorage 3키, 접근은 save.ts 어댑터로만.
- 왜: 웹 MVP 는 서버 불필요. 스팀판에서 저장소만 갈아끼우기 위해 인터페이스를 고정.
- 버린 대안: Supabase(과한 장비), localStorage 직접 호출(스팀 이식 때 전체 수술).
- 영향: 게임 코드에 localStorage 문자열이 등장하지 않는다.

## 2026-07-07 PC 가로 우선, 자급자족 에셋
- 결정: PC 가로 기준으로 설계(모바일 대응 유지), 외부 폰트·이미지 0.
- 왜: 스팀 지향(PC 화면). 외부 의존이 없으면 빠르고 오프라인·심의에도 유리.
- 버린 대안: 모바일 세로 우선(웹 킬링타임 기준), 구글폰트.
- 영향: 레이아웃 기준 해상도 1280x800, 심볼은 이모지·CSS.

## 2026-07-07 기존 퀴즈형 프로토타입 폐기, 부품 재활용
- 결정: 인터뷰 전에 임시로 만든 8문답 퀴즈형 /game 은 폐기하고, 긁기 캔버스·사운드 합성·릴 애니메이션 코드는 새 게임에 재활용한다.
- 왜: 인터뷰 결과 방향이 "순수 운빨 도파민 머신"으로 확정 — 퀴즈는 스코프 밖.
- 버린 대안: 퀴즈 모드를 서브 모드로 유지(스코프 비대).
- 영향: app/game/ 라우트 삭제, ScratchCard·juice·릴 로직은 새 구조로 이식. 변경 제안서(2026-07-07-행운의8문답게임.md)는 폐기 표시.

## 2026-07-07 배포: 새 Netlify 사이트
- 결정: 게임은 새 Netlify 사이트(게임 전용 URL)로 배포한다.
- 왜: 기존 joharim-saju-planner 사이트 이름·내용과 충돌하지 않게.
- 버린 대안: 기존 사이트 덮어쓰기, Vercel.
- 영향: Netlify 사이트 생성 1회 필요(배포 시점에). 오늘 범위는 로컬 완성까지.

## 2026-07-07 라운드 마감(뱅킹) 선택형 + 이름 럭키런 확정 (허점 라운드)
- 결정: 목표 조기 달성 시 "라운드 마감" 버튼 — 마감하면 남은 기회당 보너스(25 x 라운드), 계속하면 리스크. 이름은 럭키런(Lucky Run).
- 왜: 기회 소진 강제 플레이는 목표 달성 후에도 죽을 수 있어 억울함 — 선택권을 주면 그게 곧 전략이 된다. 이름은 런 구조 정체성 + 도메인 적합성.
- 버린 대안: 달성 즉시 클리어(욕심 재미 소멸), 강제 소진(억울함), 잭팟런/긁긁카지노.
- 영향: HUD 에 마감 버튼, 보너스 산식 rules.ts, 타이틀 로고·Netlify 사이트명 lucky-run.
